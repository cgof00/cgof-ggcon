# Pasta Public - Logos

Coloque os seguintes arquivos nesta pasta:

## Logo da Tela de Login
- **Arquivo**: `logo.png`
- **Uso**: Exibido na tela de login (altura: 96px)
- **Dimensão recomendada**: 300x96px ou similar

## Logo do Cabeçalho
- **Arquivo**: `logo1.png`
- **Uso**: Exibido no cabeçalho do sistema (altura: 40px)
- **Dimensão recomendada**: 120x40px ou similar

Os logos devem estar em formato PNG com fundo transparente (PNG-8 ou PNG-24).
Se não encontrar os arquivos, os ícones padrão serão exibidos.
